📌 常用比赛指令

前缀支持使用 . 或 !

.ready / .r将玩家标记为准备就绪
.unready / .ur取消准备状态
.tac发起战术暂停
.stop双方输入可恢复当前回合备份
.stay / .switch刀人局获胜后选择阵营

🛠️ 练习模式专用 (.prac)

输入 .prac 开启练习模式后方可使用

.rethrow / .rt重新投掷上一个投掷物
.bot / .nobots添加或移除机器人
.god / .noflash开启无敌或防闪光干扰
.traj开启/关闭投掷物轨迹预览
.clear清除场上所有的烟雾和火焰
.last传送到上一次投掷的位置

🛡️ 管理员维护指令

.start / .restart强制开始或重置比赛
.map <地图名>更换服务器当前地图
.roundknife开启/关闭刀人局模式
.rcon <命令>向服务器发送控制台指令
